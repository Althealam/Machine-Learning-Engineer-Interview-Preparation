# ✅ Machine Learning Knowledge Checklist

> This checklist summarizes the **core machine learning knowledge** expected from a  
> **Machine Learning Engineer**.  
>  
> It is designed for **interview preparation**, **self-assessment**, and **systematic review**.


## 1️⃣ Loss Functions & Objective Design

### Regression
- [☑️] Mean Squared Error (MSE)
- [☑️] Mean Absolute Error (MAE)
- [☑️] Huber Loss
- [❓] Relationship between loss functions and **noise assumptions**

### Classification
- [☑️] Binary Cross-Entropy (Log Loss)
- [☑️] Categorical Cross-Entropy (Softmax + CE)
- [❓] Difference between **multiclass** and **multilabel** classification
- [☑️] Why **cross-entropy is preferred over MSE** for classification

### Ranking & Retrieval
- [☑️] Pointwise vs Pairwise vs Listwise ranking
- [☑️] Hinge loss for ranking
- [☑️] Pairwise logistic loss (BPR / RankNet)
- [☑️] InfoNCE loss and its relationship to cross-entropy
